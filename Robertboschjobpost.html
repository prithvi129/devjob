Robert Bosch Recruitment 2023 | Entry Level | Core Java Software Engineer | BE/ B.Tech/ ME/ M.Tech – CSE/ IT/ ECE | Bangalore


Experience: Entry Level

Designation: Core Java Software Engineer

Industry: IT/Computers - Software

Job Location: Bangalore

Company Website: Robert Bosch.com

Company Robert Bosch India :

Bosch is a top provider of goods and services in the fields of energy and structure technology, consumer goods, artificial technology, and mobility results. Bosch also has the largest development installation outside of Germany in India, where it develops complete engineering and technological results. Bosch Limited, Bosch Chassis Systems India Limited, Bosch Rexroth India Limited, Bosch Engineering and Business results Private Limited, Bosch Automotive Electronics India Private Limited, Bosch Electrical Drives India Private Limited, BSH Home Appliances Private Limited, ETAS Automotive India PrivateLtd., and Robert Bosch Automotive Steering IndiaPvt.Ltd. are the nine companies under which the Bosch Group operates in India.

Eligibility Criteria for Robert Bosch Recruitment 2023:

  • BE/ B.Tech/ ME/M.Tech in Electronics/ Computer Science.
  • Strong programming and debugging skill.

Job Requirements:

  • The Engineer – Core Java is responsible for Application & Middleware development in Java for embedded connectivity products. Projects could include: Connectivity Modules and Network Gateway Modules, utilizing technologies such as embedded microprocessors, Ethernet, cellular modems (LTE, 5G, etc..), Bluetooth / WiFi chipsets along with GPS/GNSS technology. Function of the role is to deliver production intent, automotive-grade software to enable connectivity and security for future vehicles and mobility solutions. This position is located in our Bangalore, India Office. The following Skillset is necessary to be effective in this position: Proficiency in Java, OOPS-based software development- Proficiency in device side JAVA software development- Proficiency in cyber security and security certification concepts. Working experience with Linux-based systems. Should be able to independently handle Continuous Integration (CI) and Continuous Testing (CT) activities. Should have a fair understanding of SDLC life cycle. Sound knowledge and working experience on Embedded software and systems. Overview and working experience on IoT ecosystem. Strong software debugging and analytical skills. Should be able to work as a team member in an agile methodologies-based environment Strong communication skills, and Japanese language skills would be an advantage.
  • 1. Strong programming and debugging skills
  • Core Java (Must)
  • OOPS Concepts (Must)
  • C++ (Optional)
  • C (Optional)
  • 2. Knowledge of networking concepts
  • TCP/IP
  • HTTPS
  • UDP
  • MQTT
  • 3. Web services concept and usage
  • 4. Problem Solving skills
  • 5. Experience on complete SDLC and agile phases on the project
